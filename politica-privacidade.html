POLÍTICA DE PRIVACIDADE - LEQUE DE MODAS

Última atualização: 30 de agosto de 2025
Versão do aplicativo: 1.0.12

1. INFORMAÇÕES COLETADAS

Dados fornecidos pelo usuário:

  • Nome completo
  • Endereço de email
  • Foto de perfil (opcional)
  • Informações de contato

Dados coletados automaticamente:

  • Histórico de pedidos e favoritos
  • Dados de uso do aplicativo
  • Preferências e configurações
  • Informações de dispositivo (modelo, sistema operacional, versão do app)
  • Dados de localização (apenas se autorizado pelo usuário)

Dados de terceiros:

  • Informações de lojas parceiras
  • Catálogo de produtos
  • Dados de empreendimentos

2. COMO USAMOS OS DADOS

Utilizamos suas informações para:

  • Fornecer e melhorar nossos serviços de catálogo de moda
  • Processar pedidos e facilitar comunicação com lojistas
  • Personalizar sua experiência de navegação
  • Enviar notificações sobre pedidos e atualizações
  • Analisar o uso do aplicativo para melhorias
  • Cumprir obrigações legais
  • Prevenir fraudes e garantir segurança

3. COMPARTILHAMENTO DE DADOS

Compartilhamos dados com:

  • Lojistas parceiros: Para processamento de pedidos
  • WhatsApp: Para comunicação sobre pedidos
  • Prestadores de serviços: Hospedagem e análise técnica
  • Autoridades competentes: Quando exigido por lei
Importante: Nunca vendemos seus dados pessoais para terceiros.

4. SEGURANÇA DOS DADOS

Implementamos medidas de segurança robustas:

  • Criptografia de dados sensíveis
  • Acesso restrito a informações pessoais
  • Monitoramento de segurança 24/7
  • Backup regular dos dados
  • Treinamento da equipe em proteção de dados

5. RETENÇÃO DE DADOS

  • Dados de pedidos: 5 anos
  • Dados de conta: Até solicitação de exclusão
  • Dados de uso: 2 anos
  • Dados de localização: Apenas durante o uso do app

6. SEUS DIREITOS

Você tem o direito de:

  • Acessar suas informações pessoais
  • Corrigir dados incorretos
  • Solicitar exclusão de dados
  • Limitar o processamento
  • Portabilidade de dados
  • Opor-se ao processamento
  • Retirar consentimento a qualquer momento

7. MENORES DE IDADE

Nosso aplicativo não é direcionado a menores de 18 anos. Não coletamos intencionalmente informações de menores.

8. TRANSFERÊNCIA INTERNACIONAL

Suas informações podem ser transferidas e processadas em países diferentes do seu, sempre com proteções adequadas conforme a legislação aplicável.

9. ALTERAÇÕES

Esta política pode ser atualizada periodicamente. Notificaremos sobre mudanças significativas através do aplicativo.

10. CONTATO

Para exercer seus direitos ou esclarecer dúvidas:

  • Email: privacidade@lequedemodas.com.br
  • WhatsApp: (11) 99999-9999

11. CONFORMIDADE

Esta política está em conformidade com:

  • Lei Geral de Proteção de Dados (LGPD) - Brasil
  • General Data Protection Regulation (GDPR) - Europa
  • Children's Online Privacy Protection Act (COPPA) - EUA
